# SmartDocChat 🤖
A production-grade RAG chatbot built with Weaviate, LangChain, and Ollama.

## Tech Stack
- Vector DB: Weaviate (hybrid search)
- Embeddings: sentence-transformers
- LLM: Ollama llama3
- Framework: LangChain

## Setup
1. Clone the repo
   (git clone https://github.com/PRANAYM-235/SmartDocChat.git)
2. Install dependencies: `pip install -r requirements.txt`
3. Create `.env` file with your Weaviate credentials
4. Run the notebook
